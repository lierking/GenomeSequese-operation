# Gadget
**利用Perl等处理常见的基因组编辑问题

* 01.Reverse

将序列进行反向互补后输出

* 02.OneLine

将多行的fasta序列输出为一行id一行序列

* 03.Extract

根据ID列表提取fasta文件中对应的序列

* 04.Max

gene.txt是包含了一些基因名的列表，编写perl脚本，找出出现次数最多的基因

* 05.Stat&GC%

统计碱基个数以及GC%

* 06.Slice

Perl定长切割字符串、切割的返回数组

* 07.Compare

统计两个文件中的共有及特有内容

* 08.Common

编写脚本，找出这两条序列中最长的共有序列。

* 09.Stat2

统计文本中每个单词出现次数并排序

* 10.Stat3

用于fasta、fastq格式文件的read数、碱基数、最长的read、最短的read及平均read长度

* 11.N50

计算fasta格式文件的N50、N90

* 12.Translate

对于给定的核苷酸序列，翻译成蛋白质
